# ADR-0001: Use Python and FastAPI

- Status: Accepted
- Date: 2026-06-27

## Context

FrameFlow needs a productive web API stack with strong ecosystem support, good typing, and easy deployment on small self-hosted servers.

## Decision

Use Python with FastAPI as the primary application framework.

## Consequences

- API development is fast and OpenAPI support is built in.
- Python image and media tooling is readily available.
- Runtime performance should be monitored as libraries grow.
