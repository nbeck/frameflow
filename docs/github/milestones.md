# Recommended GitHub Milestones

## Milestone 0: Repository Foundation

Purpose: establish the open source repository foundation before application code.

## Milestone 1: Core Domain and Storage

Purpose: implement domain model, database foundation, and storage model.

## Milestone 2: Provider Reconciliation

Purpose: implement first provider and reconciliation lifecycle.

## Milestone 3: Rotation Engine

Purpose: implement explainable selection and repetition avoidance.

## Milestone 4: DAKboard Delivery

Purpose: serve display-ready images to DAKboard.

## Milestone 5: Operations and Hardening

Purpose: improve deployment, backups, diagnostics, security, and release readiness.
